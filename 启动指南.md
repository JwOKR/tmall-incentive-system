## 🚀 手动启动步骤

### 1. 启动后端服务
```bash
cd server
npm install
npx prisma generate
npx prisma db push
npm run dev
```

### 2. 启动前端服务（新终端）
```bash
cd client
npm install
npm run dev
```

## 📊 API测试

### 获取仪表盘数据
```bash
curl http://localhost:3001/api/dashboard/stats
```

### 获取接单人列表
```bash
curl http://localhost:3001/api/takers
```

### 获取任务列表
```bash
curl http://localhost:3001/api/tasks
```

### 获取订单列表
```bash
curl http://localhost:3001/api/orders
```

## 🛠️ 常用命令

### 停止服务
- 关闭运行 `npm run dev` 的终端窗口
- 或按 `Ctrl + C`

### 重新初始化数据库
```bash
cd server
rm prisma/dev.db
npx prisma db push
npx ts-node prisma/seed.ts
```

### 查看数据库内容
```bash
cd server
npx prisma studio
```

---

## 📝 功能说明

1. **数据汇总** - 查看订单统计、金额汇总、状态分布
2. **接单人管理** - 添加、编辑、删除接单人
3. **任务管理** - 创建激励任务，分配给接单人
4. **订单明细** - 录入和管理激励订单
5. **操作日志** - 查看系统操作记录
